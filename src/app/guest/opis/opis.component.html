Nazwa projektu: Księgarnia

Opis projektu:
Strona księgarni internetowej, która oferuje zakup książek. Aplikacja jest podzielona na dwie główne części: część frontendową, która została zbudowana przy użyciu frameworka Angular, oraz część backendową, która wykorzystuje technologię Spring Boot.

Część frontendowa (Angular):

  • Strona główna: Użytkownicy mogą przeglądać książki do kupienia.
  • Administracja:
  • Dodawanie książek: Administratorzy mogą dodać nowe książki podając tytuł, autora, cenę i opis. Po dodaniu książki zostanie ona
  • wyświetlona na stronie głównej.
  • Edycja książek: Administratorzy mogą edytować istniejące książki.
  • Usuwanie książek: Administratorzy mogą usunąć książkę.
  • Przeglądanie historii zakupów: Zwykli użytkownicy w swoim profilu mogą przeglądać zakupione książki.

Część backendowa (Spring Boot):

  • RESTful API: Backend dostarcza RESTful API do obsługi operacji na książkach, takich jak dodawanie, edycja i usuwanie książek.
  • Baza danych: Dane o zadaniach są przechowywane w bazie danych PostgreSQL.
  • Zabezpieczenia: Aplikacja jest zabezpieczona, a dostęp do operacji chroniony jest przez Spring Security JWT.
  • Obsługa błędów: Backend obsługuje błędy i zwraca odpowiednie kody HTTP w przypadku niepowodzenia operacji.

Technologie użyte w projekcie:

  • Angular
  • TypeScript
  • Bootstrap
  • Java 11
  • Spring Boot 2.7.13
  • Baza danych PostgreSQL
  • Spring Security do autentykacji i autoryzacji
